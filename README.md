# Payment Gateway Microservice (Laravel)

A scalable payment gateway microservice integrating:
- Pix
- Boleto

Supporting multiple banks:
- Banco do Brasil
- Itaú
- Bradesco
- Santander

Built with focus on:
- Clean Architecture
- Event-driven design
- Asynchronous processing
- Resilience and scalability

## CI / CD
CI - Aplicado com Github actions e realiza testes unitários com PHPUnit, arquivos na pasta .github/workflows e no arquivo é separado as etapas de andamento
